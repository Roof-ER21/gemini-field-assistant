### Desktop (>1024px)
- 2-column grid for goal cards
- Full-width chart
- 3-column stats grid
- 2-column quick actions

### Tablet (768-1024px)
- 2-column grid (stacks on smaller tablets)
- Full-width chart
- 3-column stats grid
- 2-column quick actions

### Mobile (<768px)
- Single column layout
- Smaller circular progress (160px)
- Stacked stats cards
- Single column quick actions
- Reduced padding/spacing
- Fluid typography (clamp)
