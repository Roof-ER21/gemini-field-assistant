## Troubleshooting

### "Susan isn't responding"
- Check your internet connection
- Refresh the page
- Try a different message
- Contact admin if persists

### "Voice input not working"
- Allow microphone permissions in browser
- Check microphone is connected/working
- Try clicking the mic button again
- Use text input as backup

### "Response seems incorrect"
- Provide more context in your question
- Ask Susan to clarify or explain differently
- Verify critical information independently
- Report persistent issues to admin
