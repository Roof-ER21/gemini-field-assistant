## Performance Optimization

### Spatial Query Optimization

```sql
-- Indexed spatial query
SELECT *,
  calculate_distance_miles($1, $2, latitude, longitude) as distance
FROM storm_lookups
WHERE calculate_distance_miles($1, $2, latitude, longitude) <= $3
ORDER BY distance ASC;

-- Uses indexes:
-- 1. idx_storm_lookups_location (lat, lng)
-- 2. Haversine calculation is a fast PostgreSQL function
```

### JSONB Query Optimization

```sql
-- GIN index on storm_events enables fast JSONB searches
SELECT * FROM storm_lookups
WHERE EXISTS (
  SELECT 1 FROM jsonb_array_elements(storm_events) AS event
  WHERE event->>'eventType' = 'hail'
    AND (event->>'magnitude')::numeric > 1.5
);

-- Uses: idx_storm_lookups_events (GIN index)
```
