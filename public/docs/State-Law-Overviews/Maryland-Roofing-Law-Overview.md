# Maryland Roofing Law Overview (Quick Reference)

Audience: S21 (Susan) and Roof-ER field reps. Use as quick, non-legal guidance. Confirm with MIA/AHJ.

Key Points
- Building Code: 2021 code baseline (local adoption applies)
- Matching: Maryland Insurance Administration (MIA) guidance recognizes matching considerations; use in claims where appropriate.
- Strategy: Matching arguments can be effective; also use repairability, code compliance, and manufacturer documentation.
- Permits: Generally required. Check local permit office requirements.
- Licensing: MHIC license (Home Improvement) required for residential contracting.

Use Cases
- Partial approvals: Matching + code compliance + manufacturer constraints are valid MD arguments. Include clear photo documentation.
- Documentation: Include model/color discontinuation, line width differences, and code references where applicable.

Disclaimer: This is general guidance. Confirm with MIA publications, local officials, and policy language.
