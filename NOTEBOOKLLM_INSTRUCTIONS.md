# How to Create a Video Guide with NotebookLM

## What You'll Need
1. The S21_ROOFER_APP_GUIDE.md file (created in this directory)
2. Access to NotebookLM: https://notebooklm.google.com/

## Step-by-Step Instructions

### Step 1: Open NotebookLM
1. Go to https://notebooklm.google.com/
2. Sign in with your Google account
3. Click "Create new notebook" or "+ New"

### Step 2: Upload the Guide
1. Click "Add source" or "Upload"
2. Select the file: `S21_ROOFER_APP_GUIDE.md`
3. Wait for NotebookLM to process the document
4. You'll see a summary and key topics extracted

### Step 3: Generate Audio Overview
1. Look for the "Audio Overview" or "Generate" button
2. Click "Generate Audio Overview"
3. NotebookLM will create a podcast-style conversation between two AI hosts
4. This usually takes 2-5 minutes to generate

### Step 4: Review the Audio
1. Listen to the generated audio overview
2. The AI hosts will discuss all the features conversationally
3. They'll cover:
   - What S21 ROOFER is
   - All 10 core features
   - How to use each feature
   - Best practices
   - Workflows
   - Tips and tricks

### Step 5: Download the Audio
1. Click the download button (usually a download icon)
2. Save the MP3 file
3. File name will be something like: "NotebookLM_Audio_Overview.mp3"

### Step 6: Create Video (Optional)
If you want to create a video from the audio:

**Option A: Use Video Editing Software**
1. Open video editor (iMovie, Premiere, Final Cut, DaVinci Resolve)
2. Import the NotebookLM audio file
3. Add visuals:
   - Screenshots of the S21 ROOFER app
   - Screen recordings showing features
   - Logo and branding
   - Text overlays for key points
4. Export as MP4

**Option B: Use Online Tools**
1. Use tools like:
   - Canva (free video maker)
   - Kapwing
   - Descript
   - Pictory.ai
2. Upload the audio
3. Add auto-generated captions
4. Add visuals/screenshots
5. Export

**Option C: Simple Slideshow**
1. Create PowerPoint/Keynote with screenshots
2. Add the audio as narration
3. Export as video

## What NotebookLM Will Create

The audio overview will be a **conversational podcast** between two AI hosts discussing:

### Introduction (2-3 minutes)
- What is S21 ROOFER?
- Who is it for?
- Why was it created?
- Overview of capabilities

### Feature Deep Dives (15-20 minutes)
Each feature explained conversationally:
1. Susan Chat - AI assistant
2. Email Generator - Professional emails
3. Document Analysis - Photo/doc analysis
4. Voice Transcription - Meeting recordings
5. Insurance Database - Quick reference
6. Live Conversation - Real-time help
7. Knowledge Base - Company docs
8. Agnes - Objection handler
9. Document Job Notes - Note-taking
10. Admin Panel - Analytics

### Workflows & Best Practices (5-7 minutes)
- Common use cases
- Pro tips
- Daily workflows
- Efficiency strategies

### Conclusion (2-3 minutes)
- Key benefits recap
- Getting started tips
- Call to action

**Total Duration:** Approximately 25-35 minutes of conversational audio

## Tips for Best Results

### For Better Audio Quality:
- The guide is comprehensive, so the audio will be detailed
- NotebookLM creates natural, engaging conversations
- Hosts will use examples and explain complex features simply

### For Better Video:
- Take screenshots of each feature mentioned
- Record screen demos of key workflows
- Add captions for accessibility
- Include company branding
- Keep video segments aligned with audio

### Customization Ideas:
- Edit the guide before uploading to emphasize certain features
- Add specific examples from your company
- Include testimonials or success stories
- Add your company's branding guidelines

## Alternative: Multiple Shorter Videos

Instead of one long video, you could create multiple focused videos:

1. **Overview Video** (5 min)
   - What is S21 ROOFER
   - Key benefits
   - Quick feature tour

2. **Susan Chat Tutorial** (3-4 min)
   - Upload just the Susan Chat section
   - Generate audio for this feature only

3. **Email & Documentation** (4-5 min)
   - Email Generator + Document Analysis
   - Create focused content

4. **Voice Features** (4-5 min)
   - Transcription + Live Conversation
   - Voice-specific tutorials

5. **Sales Tools** (3-4 min)
   - Agnes + Insurance Database
   - Sales-focused content

To do this:
1. Create separate markdown files for each topic
2. Upload each to NotebookLM separately
3. Generate audio for each
4. Create shorter, focused videos
